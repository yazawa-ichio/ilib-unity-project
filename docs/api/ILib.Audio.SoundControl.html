    Class SoundControl

    オーディオソース等のを必要なゲームオブジェクトを確保し管理する用のクラスです。 使用しているプレイヤーが破棄されたときに自動的に削除されます。 また、Release関数を呼ぶと生成したすべてのプールを強制的に破棄します。
